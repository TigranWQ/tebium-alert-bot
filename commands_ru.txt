Команды для загрузки TeBium Alert Bot на GitHub:

1. Перейти в папку проекта:
cd C:\Prodjects\TeBium-Alert-Bot

2. Добавить все файлы:
git add .

3. Создать коммит на русском языке:
git commit -m "Первоначальный коммит: TeBium Alert Bot - Telegram бот для мониторинга системы и алертов"

4. Удалить существующий remote:
git remote remove origin

5. Добавить GitHub remote:
git remote add origin https://github.com/TigranWQ/tebium-alert-bot.git

6. Переименовать ветку в main:
git branch -M main

7. Загрузить на GitHub:
git push -u origin main

- Password: Используйте Personal Access Token (не пароль от GitHub)
